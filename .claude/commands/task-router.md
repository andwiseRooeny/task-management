# Task Router

## 사전 조건
`_task/L1_INIT.md` 파일이 존재해야 합니다.
파일이 없으면 "/task-init을 먼저 실행해주세요" 안내 후 종료.

## 워크플로우

1. `_task/L1_INIT.md` 파일 읽기
2. 업무 설명에서 키워드 분석
3. 아래 매핑 테이블 기반으로 적합한 업무 유형 3개 추천
4. 사용자가 선택하거나 직접 입력
5. `_task/L3_CONTEXT.md` 파일 생성

## 업무 유형 매핑 테이블
